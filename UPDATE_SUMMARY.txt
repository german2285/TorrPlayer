================================================================================
                          NEW FEATURES
================================================================================

1. VERSION PARAMETER (-v|--version)
   - Specify version for builds: -v v1.0.0
   - Creates versioned ZIP packages
   - Supports semver: v1.0.0, v1.0.0-beta, v1.0.0-dev

2. GITHUB RELEASE AUTOMATION (-r|--release)
   - Automatic GitHub release creation
   - Uploads ZIP package as release asset
   - Creates tags automatically
   - Dev builds create pre-releases

3. CUSTOM RELEASE NOTES (-n|--notes)
   - Add custom release notes: -n "Release description"
   - Default notes if not provided
   - Multiline notes supported

4. HELP SYSTEM (-h|--help)
   - Built-in help for both scripts
   - Usage examples
   - Parameter descriptions
   - Link to documentation

5. ENHANCED VALIDATION
   - Checks for GitHub CLI (gh)
   - Verifies authentication
   - Validates required tools
   - Clear error messages

6. PACKAGE AUTOMATION
   - Auto-creates ZIP packages
   - Includes exe, dll, README, LICENSE
   - Proper naming: TorrPlayer-v1.0.0-windows-amd64.zip
   - Dev builds include DEV-BUILD-NOTICE.txt

================================================================================
                          USAGE EXAMPLES
================================================================================

SIMPLE BUILD:
  bash scripts/build.sh

BUILD WITH VERSION:
  bash scripts/build.sh -v v1.0.0

FULL GITHUB RELEASE:
  bash scripts/build.sh -v v1.0.0 -r -n "Release notes"

DEVELOPMENT BUILD:
  bash scripts/build-dev.sh

DEV PRE-RELEASE:
  bash scripts/build-dev.sh -v v1.0.0-dev -r

SHOW HELP:
  bash scripts/build.sh --help

================================================================================
                          PREREQUISITES
================================================================================

REQUIRED:
  ✓ Go 1.23+
  ✓ Node.js 16+
  ✓ Wails CLI
  ✓ MinGW-w64 (for Linux cross-compilation)

FOR GITHUB RELEASES:
  ✓ GitHub CLI (gh)
  ✓ GitHub authentication (gh auth login)

INSTALLATION:
  # Wails
  go install github.com/wailsapp/wails/v2/cmd/wails@latest

  # GitHub CLI
  sudo apt install gh

  # MinGW-w64
  sudo apt-get install mingw-w64
